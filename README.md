# Predictive Analysis of Stress and Mental Health Factors

This project explores the relationships between sleep habits, stress levels, physical activity, and mental health-related factors using data analysis, visualizations, and statistical analysis.

The analysis focuses on identifying possible connections between sleep quality, stress levels, physical activity, and sleep disorders through exploratory data analysis and statistical testing.

## Analysis Process

- Load and inspect the dataset.
- Check for missing values and data quality issues.
- Clean and prepare the data for analysis.
- Explore the main variables and relationships in the dataset.
- Create visualizations to better understand the data.
- Perform statistical analysis using correlation tests and t-tests.
- Summarize the main observations from the analysis.
