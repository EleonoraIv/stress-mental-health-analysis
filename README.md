# Predictive Analysis of Stress and Mental Health Factors

This project focuses on analyzing the relationship between stress, sleep quality and mental health-related factors using data related to lifestyle and workplace mental health.

## Datasets

1. `Sleep_health_and_lifestyle_dataset.csv` - https://www.kaggle.com/datasets/uom190346a/sleep-health-and-lifestyle-dataset

2. `survey.csv` - https://www.kaggle.com/datasets/osmi/mental-health-in-tech-survey
